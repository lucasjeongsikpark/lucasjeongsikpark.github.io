Multi-Agent Evaluation System

LLM-as-a-Judge and Multi-Agent Debate Evaluation

Aug 2025 -- Nov 2025

Report Preview

Overview

This project surveys Multi-Agent Debate Evaluation Frameworks (MADEF) and evaluates them as a scalable alternative to human judgment. It compares four representative MADEF systems across open-domain QA, multi-step reasoning, and medical QA.

Technical Approach

The report examines how multiple LLM agents debate and judge AI-generated outputs, then aggregate rubric scores into a final decision. It focuses on robustness, interpretability, and the cost of using local open-weight judge models.

Evaluation Setup

Experiments compare Debatable-Intelligence, MORE, DEBATE, and ChatEval using standardized prompts, rubrics, and datasets. The setup measures judge consistency, latency, score variance, and alignment with human judgments.

Results / Observations

  • MADEFs can approximate human judgment across several domains while varying significantly in cost and computational complexity.
  • Multi-agent debate exposes reasoning traces that improve interpretability compared to single-agent judge approaches.
  • More complex MADEFs do not always outperform lightweight configurations, highlighting a practical tradeoff between robustness and efficiency.
